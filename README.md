# AI Speech Interview Scorer

A full-stack AI-powered application that generates role-specific interview questions tailored to job roles and user input, records spoken answers, and scores them using AI.

## Features

- 🎤 Audio recording uploads and URL-based audio ingestion for speech analysis
- 🤖 AI-generated interview questions tailored to job roles and user input
- 📊 Automatic scoring and feedback powered by OpenAI
- 🔐 Secure user authentication including login, registration, and session management
- 🗄️ MongoDB database integration for data persistence

## Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Next.js API Routes
- **Database:** MongoDB
- **AI:** OpenAI API
- **Auth:** NextAuth.js

## Getting Started

1. Clone the repo
```bash
   git clone https://github.com/mohamed-bah/AI-Speech-Interview-Scorer-.git
```

2. Install dependencies
```bash
   npm install
```

3. Create a `.env.local` file and add your environment variables
